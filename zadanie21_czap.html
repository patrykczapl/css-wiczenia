Identyfikator ID



Pojęcie Znaczenie
Co znaczy skrót ID? ID to skrót od identyfikatora. Jest to unikalny atrybut, który można przypisać do danego elementu HTML.
Jaki jest cel stosowania ID? Cel stosowania ID to zróżnicowanie podobnych elementów na stronie. Dzięki ID możemy przypisać specyficzne formatowanie do konkretnego elementu.
Jakie jest znaczenie znaku #? Znak # jest używany przed nazwą identyfikatora podczas jego definiowania w CSS.
Jak następuje wywołanie ID? Wywołanie identyfikatora następuje poprzez użycie atrybutu id w elemencie HTML. Na przykład, jeśli zdefiniowaliśmy ID w CSS jako #pochyly, możemy go użyć w elemencie HTML tak:

Uwagi na temat stosowania ID?
  • Różnica między id a class -> Identyfikator może wystąpić tylko raz w całym dokumencie, natomiast klasę można powielać do woli.
  • Podobnie jak klasy, identyfikatorów, możemy używać tyle razy, ile razy uznamy to za konieczne. Działanie takie jest błędem, ale nie wpływa na poprawność działania strony.
  • Elementy oznaczone ID muszą być unikatowe, tzn., że na jednej stronie może wystąpić tylko 1 element z takim ID. Jeśli nie mamy pewności, czy element wystąpi więcej niż raz to możemy bez obaw użyć 'class', które może być przypisane do nieskończenie wielu elementów na stronie.
